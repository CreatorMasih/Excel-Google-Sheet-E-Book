Cell References — 3 Types

💡 Relative Reference

=A1+B1 — Jab formula copy karo to reference apne aap change hota hai. Jaise row 2 pe copy kiya to =A2+B2 ho jaata hai.

📌 Absolute Reference — $ sign

=$A$1+B1 — $ lagane se reference lock ho jaata hai. Copy karne pe $A$1 kabhi nahi badle ga.

Shortcut: Cell select karo → F4 dabao

⚡ Mixed Reference

=A$1 — Row lock, column free
=$A1 — Column lock, row free

Multiplication tables banane mein kaam aata hai!

🎯 Common Interview Questions
Absolute aur Relative reference mein kya difference hai?
Relative reference copy karne pe change hota hai (A1 → A2), jabki Absolute reference $ sign ki wajah se lock rehta hai ($A$1 hamesha $A$1 rehta hai). Tax rate, commission rate jaise fixed values ke liye $ use karte hain.
Worksheet aur Workbook mein kya antar hai?
Workbook poori Excel file hoti hai (.xlsx), jaise ek notebook. Worksheet us notebook ka ek page hota hai (Sheet1, Sheet2). Ek workbook mein maximum 255 worksheets ho sakti hain.
⚠️ Common Mistake

Bahut log $ sign bhool jaate hain aur formula copy karne pe galat results aate hain. Tax, commission ya fixed multiplier use karte waqt hamesha F4 key se absolute reference lagao!

Basic Formulas — Har Formula Deeply Samjho

SUM se lekar RANDBETWEEN tak — sab kuch real examples ke saath

📌 Formula Likhne ka Rule

Har formula = (equal sign) se shuru hota hai. Jaise: =SUM(A1:A10)
Agar = nahi likha to Excel use text maan leta hai.

SUM
Most Used Formula
📌 Kya hai?
Numbers ko jodhta hai (addition karta hai)
💼 Kab use?
Total salary, total attendance, total sales nikalne ke liye
📝 Syntax
=SUM(number1, number2, ...) ya =SUM(A1:A10)
📊 HR Example
=SUM(B2:B50) → 50 employees ki salary ka total
🎯 Interview Q
"SUM aur SUMIF mein kya difference hai?" → SUM sab numbers jodta hai, SUMIF condition ke saath jodta hai
⚠️ Mistake
Text cells include karne par SUM unhe ignore karta hai — careful rahein
AVERAGE
Statistical
📌 Kya hai?
Numbers ka average (mean) nikalata hai
📝 Syntax
=AVERAGE(A1:A10)
📊 HR Example
=AVERAGE(B2:B20) → Average salary of team
⚠️ Mistake
Empty cells ignore hoti hain — zeros count nahi hote. Sahi average ke liye zeros daalo!

🎯 Interview Questions — Basic Formulas
COUNT aur COUNTA mein kya antar hai?
COUNT sirf numbers count karta hai, text ya blank cells ignore karta hai. COUNTA koi bhi non-empty cell count karta hai — chahe text ho, number ho, date ho. Employee names count karne ke liye COUNTA use karte hain.
INT aur ROUNDDOWN mein kya difference hai?
Positive numbers ke liye same result aata hai. Lekin negative numbers mein difference hai: INT(-4.7) = -5, jabki ROUNDDOWN(-4.7, 0) = -4. Isliye salary calculations mein ROUNDDOWN safe hai.

📝 Practice Questions

  1. 50 employees ki salary ka total aur average nikalo
  2. Highest aur lowest salary find karo
  3. PF = Basic × 12% — nearest rupee mein calculate karo using ROUND
  4. Kitne employees ka joining date fill hai? (COUNTA use karo)
  5. Kitne employees ka department missing hai? (COUNTBLANK)
Chapter 4

Logical Functions — Decision Making

IF, IFS, AND, OR, IFERROR — har condition ko handle karo

IF
⭐ Most Important
📌 Kya hai?
Ek condition check karta hai — agar true to ek value, agar false to doosri value
📝 Syntax
=IF(condition, value_if_true, value_if_false)
📊 HR Example
=IF(B2>=35000,"Senior","Junior") → 35k+ salary wale Senior
📊 Attendance
=IF(C2>=80,"Eligible","Not Eligible") → 80% attendance pe eligible
⚠️ Mistake
Text hamesha quotes mein likho: "Senior" — quotes na ho to error aata hai

Nested IF — IF ke andar IF

⚠️ Nested IF Kyun Use Karte Hain?

Jab ek se zyada conditions ho. Jaise salary grade A, B, ya C decide karna ho.

📝 Nested IF Syntax
=IF(B2>=50000,"Grade A",
  IF(B2>=35000,"Grade B",
    IF(B2>=20000,"Grade C","Grade D")))

B — Salary

IFS — Nested IF Ka Better Version

💡 IFS Function (Excel 2019+, Google Sheets)

=IFS(B2>=50000,"Grade A", B2>=35000,"Grade B", B2>=20000,"Grade C", TRUE,"Grade D")

Zyada readable hai — sabhi conditions ek line mein. TRUE last condition default ke liye use karo.

AND
Logical
📌 Kya hai?
Sabhi conditions true ho tab TRUE return karta hai
📝 Syntax
=AND(cond1, cond2, ...)
📊 Example
=IF(AND(B2>=35000, C2="Active"),"Bonus","No Bonus")
OR
Logical
📌 Kya hai?
Koi bhi ek condition true ho tab TRUE return karta hai
📝 Syntax
=OR(cond1, cond2, ...)
📊 Example
=IF(OR(C2="HR",C2="Admin"),"Office","Field")
IFERROR
⭐ Error Handler
📌 Kya hai?
Agar formula mein error aaye to custom message dikhata hai — #N/A, #DIV/0!, #VALUE! etc. hide karta hai
📝 Syntax
=IFERROR(formula, "Error message")
📊 Example
=IFERROR(VLOOKUP(A2,Sheet2!A:B,2,0),"Not Found")
💡 Pro Tip
VLOOKUP ke saath hamesha IFERROR wrap karo — professional reports mein #N/A dikhna accha nahi lagta

SWITCH — Multiple Conditions Clean Way

📝 SWITCH Syntax (Google Sheets & Excel 2019+)
=SWITCH(expression, val1, result1, val2, result2, ..., default)

=SWITCH(C2,"HR","Human Resources","IT","Information Tech","Sales","Sales Dept","Other")
↑ Department code ko full name mein convert karta hai
🎯 Interview Questions
IF aur IFS mein kya difference hai?
IF mein ek condition check hoti hai, aur nested IF se multiple conditions check karte hain — lekin yeh complex aur error-prone ho jaata hai. IFS ek modern function hai jo multiple conditions ek saath cleanly handle karta hai — zyada readable aur easy to maintain.
IFERROR kyun use karte hain?
Jab VLOOKUP ya koi formula error (#N/A, #DIV/0!) deta hai, to IFERROR use karke hum "Not Found" ya "" jaise clean messages dikhate hain. Yeh professional reports ke liye zaroori hai.

🎯 Interview Questions
TRIM function kab use karte hain?
Jab data kisi system se copy-paste kiya ho ya Google Forms se aaya ho — usme extra spaces hote hain. TRIM se leading, trailing aur extra middle spaces hat jaate hain. VLOOKUP galat result de raha ho tab bhi TRIM lagao — aksar spaces ki wajah se hota hai.
CONCATENATE aur TEXTJOIN mein kya difference hai?
CONCATENATE sirf cells jodta hai bina separator ke. TEXTJOIN mein aap separator define kar sakte ho (jaise space ya comma) aur empty cells ignore karne ka option bhi hai. Modern Excel mein TEXTJOIN prefer karein.

NETWORKDAYS — Leave aur Working Days

📊 Real HR Use

Situation: Employee ne 1-Jan se 31-Jan tak kaam kiya. Kitne working days hain?

=NETWORKDAYS("01-Jan-2025","31-Jan-2025") → 23 days

Holidays ko exclude karne ke liye: =NETWORKDAYS(start, end, holidays_range)

Employee ka experience calculate kaise karein?

Chapter 7

Lookup Functions — Data Dhundna

VLOOKUP, XLOOKUP, INDEX+MATCH — interviews mein sabse zyada pooche jaate hain!

⚠️ Most Important Chapter!

Har MIS, HR Ops, aur Data Analyst interview mein VLOOKUP zaroor poochhte hain. Yeh chapter bahut dhyan se padhein!

VLOOKUP
⭐ #1 Most Asked
📌 Kya hai?
Ek table mein vertically (column mein) value search karta hai aur same row se related value return karta hai
📝 Syntax
=VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])
Parameters
lookup_value = Jo dhundna hai (Employee ID)
table_array = Poori table (A:D)
col_index_num = Kitni column mein answer hai (2 = 2nd column)
range_lookup = 0 ya FALSE = exact match (hamesha 0 likho!)
📊 HR Example
=VLOOKUP(A2, Sheet2!A:D, 3, 0) → Employee ID se Department dhundho
⚠️ Limitations
• Sirf left se right search karta hai
• Table mein column order change nahi kar sakte
• Left mein value search nahi kar sakta

XLOOKUP
Modern — Excel 365+
📌 Kya hai?
VLOOKUP ka advanced version — left, right, dono directions mein search kar sakta hai
📝 Syntax
=XLOOKUP(lookup_value, lookup_array, return_array, [if_not_found], [match_mode], [search_mode])
📊 Example
=XLOOKUP(A2, Sheet2!A:A, Sheet2!C:C, "Not Found")
✅ Advantages
Left search possible | Column number nahi likhna | Built-in "not found" value | Faster

INDEX + MATCH — Power Combo

💡 INDEX + MATCH Kyun Seekhein?

VLOOKUP sirf right direction mein dhundta hai. INDEX+MATCH kisi bhi direction mein dhundta hai — left bhi!

📌 INDEX — Row Column se Value

=INDEX(array, row_num, col_num)

Table mein ek specific cell ki value nikalta hai position ke basis pe.

=INDEX(A1:D50, 3, 2) → 3rd row, 2nd column ki value

📌 MATCH — Position Dhundna

=MATCH(lookup_val, lookup_array, match_type)

Ek value ki position (row number) return karta hai array mein.

=MATCH("E003", A1:A50, 0) → E003 kaunsi row mein hai

📝 INDEX + MATCH Combined Formula
=INDEX(C:C, MATCH(F2, A:A, 0))
↑ F2 mein Employee ID dalo → C column se Department milega — left column bhi search ho sakta hai!

🎯 Interview Questions — Lookup
VLOOKUP kya hai aur kaise use karte hain? Ek example do.
VLOOKUP ek lookup formula hai jo table mein vertically search karke related value return karta hai. Jaise: Employee Master Sheet mein Employee ID se Department dhundna: =IFERROR(VLOOKUP(A2, MasterSheet!A:D, 3, 0), "Not Found") — A2 mein ID hai, MasterSheet mein search karo, 3rd column (Department) return karo, 0 = exact match, IFERROR se #N/A avoid karo.
VLOOKUP ki kya limitations hain?
1. Sirf right direction mein search karta hai — left column se value nahi la sakta. 2. Column number hardcode karna padta hai — column add/delete hone pe formula break hota hai. 3. Large data mein slow hota hai. Isliye XLOOKUP ya INDEX+MATCH better hain.
INDEX MATCH VLOOKUP se better kyun hai?
INDEX+MATCH kisi bhi direction mein search kar sakta hai (left bhi), column number change hone pe break nahi hota, aur large datasets mein VLOOKUP se fast hai. Yeh combination zyada flexible aur professional hai.

150+ Interview Questions with Answers

Basic se Advanced — sab levels ke sawaal yahan hain!

🟢 Basic Level Questions

Excel mein formula kaise likhte hain?
Har formula = (equal sign) se shuru hota hai. Jaise =SUM(A1:A10) ya =VLOOKUP(A2,B:D,2,0). Bina = ke Excel use text maanta hai.
Cell address kya hota hai? Ek example do.
Cell address row aur column ka combination hota hai — pehle column letter, phir row number. Jaise A1 (Column A, Row 1), B5 (Column B, Row 5), C10. Yeh Name Box mein dikhta hai.
Relative aur Absolute reference mein difference?
Relative (A1) copy karte waqt change hota hai. Absolute ($A$1) $ sign se lock hota hai — copy karne pe nahi badle ga. F4 se $ lagao. Tax rate, fixed multiplier ke liye absolute use karo.
SUM formula kaise likhein?
=SUM(A1:A10) — A1 se A10 tak sab numbers jodega. Ya =SUM(A1,B1,C1) — individual cells. Ya =SUM(A:A) — poori A column.
Excel mein data ko kaise sort karein?
Data select karo → Data tab → Sort → Column choose karo → A to Z ya Z to A → OK. Ya header row mein filter lagao (Ctrl+Shift+L) aur dropdown se sort karo.
Freeze Panes kyun use karte hain?
Bade data mein scroll karte waqt headers scroll ke saath chale jaate hain. Freeze karne se header hamesha visible rehta hai. View → Freeze Panes → Freeze First Row.
TRIM kya karta hai?
=TRIM(A2) text se extra spaces remove karta hai — leading, trailing, aur multiple middle spaces. Data copy-paste ya form submission se aaye data mein aksar extra spaces hote hain — TRIM se saaf hota hai.
PROPER function kya karta hai?
=PROPER("rahul sharma") → Rahul Sharma. Har word ka pehla letter capital, baaki small karta hai. Employee names aur addresses ke liye use hota hai.
How to remove duplicates in Excel?
Data select karo → Data tab → Remove Duplicates → Kaunsi columns check karni hain wo select karo → OK. Note: Pehle original data ka backup rakho!
Excel mein filter kaise lagate hain?
Header row mein koi bhi cell click karo → Data → Filter (ya Ctrl+Shift+L) → Har column mein dropdown arrow aata hai → Click karke filter choose karo.

🟡 Intermediate Level Questions

VLOOKUP explain karo with example.
VLOOKUP vertically search karke related value return karta hai. Syntax: =VLOOKUP(lookup_value, table_array, col_index, range_lookup). Example: Employee ID se department dhundna: =IFERROR(VLOOKUP(A2, MasterSheet!A:D, 3, 0), "Not Found") — A2=ID, A:D=table, 3=3rd column=Department, 0=exact match, IFERROR handles #N/A.
IF function ek HR example ke saath explain karo.
=IF(C2>=80%, "Eligible", "Not Eligible") — Attendance 80% ya zyada ho to "Eligible" warna "Not Eligible". Nested: =IF(B2>=50000,"Grade A",IF(B2>=35000,"Grade B","Grade C")) — salary se grade determine karna.
COUNTIF aur COUNTIFS mein difference?
COUNTIF ek condition: =COUNTIF(B:B,"HR") — HR employees count. COUNTIFS multiple: =COUNTIFS(B:B,"HR",D:D,"Active") — HR ke Active employees. COUNTIFS mein pairs mein conditions dete hain.
Employee ki age ya experience kaise calculate karein?
DATEDIF use karo: =DATEDIF(B2,TODAY(),"Y")&" Years "&DATEDIF(B2,TODAY(),"YM")&" Months" — B2 mein joining/DOB date, TODAY() end date, "Y" years, "YM" remaining months.
Pivot Table kaise banate hain?
Data mein koi bhi cell click karo → Insert → PivotTable → New Worksheet → OK → Right side field list se fields drag karo — Department to Rows, Salary to Values (Sum), Status to Filters → Report ready!
SUMIF formula explain karo.
=SUMIF(range, criteria, sum_range). Example: HR department ki total salary: =SUMIF(B:B,"HR",C:C) — B column check karo "HR" ke liye, matching rows ki C column (salary) sum karo.
IFERROR kyun lagate hain VLOOKUP ke saath?
VLOOKUP agar value nahi dhundh paata to #N/A error deta hai — jo unprofessional dikhta hai. IFERROR=IFERROR(VLOOKUP(...),"Not Found") se error ki jagah "Not Found" aata hai — reports professional lagti hain.
Conditional Formatting kya hai? Example do.
Conditional Formatting cells ko automatically color karta hai condition ke basis pe. Home → Conditional Formatting → Highlight Cell Rules → Greater Than → 50000 → Green fill. Salary column mein 50k+ wale automatically green ho jaate hain.
NETWORKDAYS formula kab use karte hain?
Working days count karne ke liye. =NETWORKDAYS("01-Jan-2025","31-Jan-2025") — January 2025 mein weekends exclude karke working days. Holidays exclude karne ke liye third argument mein holiday dates ki range do.
CONCATENATE aur & operator mein kya antar hai?
Dono cells jodne ke liye use hote hain: =CONCATENATE(A2," ",B2) = =A2&" "&B2. Lekin TEXTJOIN zyada advanced hai — separator aur empty cells ignore karne ka option deta hai: =TEXTJOIN(" ",TRUE,A2,B2,C2).

🔴 Advanced Level Questions

INDEX MATCH VLOOKUP se better kyun hai?
Teen karan: 1) Left search — VLOOKUP sirf right mein dhundta, INDEX+MATCH kisi bhi direction mein. 2) Column insert/delete hone pe VLOOKUP ka col_index break hota hai, INDEX+MATCH ka nahi. 3) Large data mein INDEX+MATCH faster hai. Formula: =INDEX(C:C,MATCH(F2,A:A,0)).
XLOOKUP kya hai aur VLOOKUP se better kyun?
XLOOKUP (Excel 365+) VLOOKUP ka modern replacement hai: 1) Left/right/up/down — kisi bhi direction mein search. 2) Built-in "if not found" parameter. 3) Multiple columns return kar sakta hai. 4) Default exact match. Syntax: =XLOOKUP(value, lookup_array, return_array, "Not Found").
QUERY function kya hai Google Sheets mein?
QUERY SQL-like language se data analyze karta hai. =QUERY(A:D,"SELECT B, SUM(C) WHERE D='Active' GROUP BY B ORDER BY SUM(C) DESC LABEL SUM(C) 'Total Salary'",1) — Active employees ki department-wise total salary descending order mein.
Pivot Table mein Calculated Field kya hota hai?
Existing pivot fields se custom formula banana — bina original data mein column add kiye. PivotTable Analyze → Fields, Items → Calculated Field → Name: "Annual PF" → Formula: =Salary*12% → Add. Ab Annual PF directly pivot mein dikhega.
Large dataset mein Excel slow ho raha hai — kya karein?
1. Volatile formulas (TODAY, NOW, RAND) kam karo. 2. VLOOKUP ki jagah INDEX+MATCH use karo. 3. Entire column references (A:A) ki jagah specific range (A2:A1000) use karo. 4. Unnecessary conditional formatting hatao. 5. Manual calculation mode on karo (Formulas → Calculation Options → Manual).
ARRAYFORMULA kya karta hai? (Google Sheets)
Ek formula ko poori range pe apply karta hai — har row mein separately likhne ki zaroorat nahi. =ARRAYFORMULA(A2:A100*B2:B100) — poore column ke products ek saath. =ARRAYFORMULA(IF(C2:C100="Active","✅","❌")) — poori column mein status icons.
IMPORTRANGE kaise use karte hain? Permission issue kaise solve karo?
=IMPORTRANGE("sheet_url","Sheet1!A1:D100"). Pehli baar use mein cell pe click karo → "Allow Access" button aayega → click karo → permission mil jaati hai. Agar permission deny ho to: source sheet owner ko access dene ke liye bolo ya Share settings check karo.

VLOOKUP #N/A error aa raha hai — kyun aur kaise fix karein?
4 common reasons: 1) Extra spaces — TRIM lagao: =VLOOKUP(TRIM(A2),...). 2) Exact match nahi mila — confirm 0 fourth argument hai. 3) Value column first mein nahi — table restructure karo ya INDEX+MATCH use karo. 4) Number text ke roop mein store — VALUE() function use karo. IFERROR se error handle karo.
Recruitment tracker Google Sheets mein banana hai jisme 5 HR log ek saath kaam karein. Aap kya recommend karoge?
Google Sheets best choice hai: 1) Shared spreadsheet banao. 2) IMPORTRANGE se HR Master se data pull karo. 3) Data Validation — Dropdown se Stage aur Status. 4) Conditional Formatting — Selected=Green, Rejected=Red. 5) Filter views — har HR apna filter save kare. 6) Pivot Table — monthly summary. 7) Google Forms se candidate info directly collect karo.
